JSON Interactive Labs (10 Labs)

Each lab has: (A) Plain text → JSON conversion with instant validation + basic structure checks, and (B) a visible JSON quiz with 10 questions.
Ground rules for valid JSON
  • Use double quotes for keys and strings
  • No trailing commas
  • Objects use { }, arrays use [ ]
  • Keep types consistent (number vs string vs boolean vs null)
